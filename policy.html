使用政策

最近更新：2026 年 4 月 18 日

1. 概述

本使用政策（Acceptable Use Policy）规定了 AnPin AI 平台服务的合理使用规范。所有用户在使用本平台时必须遵守本政策。违反本政策可能导致账户被暂停或永久终止。

2. 允许的用途

您可以将本平台用于以下合法目的：

  • 软件开发与编程辅助
  • 内容创作与文案撰写
  • 数据分析与研究
  • 教育和学习
  • 客户服务与商业运营
  • 合法的商业应用开发

3. 禁止行为

以下行为严格禁止，违反者将被立即封停账户，且不予退还余额。

3.1 违法活动

  • 生成、传播违反法律法规的内容
  • 利用 AI 进行诈骗或其他犯罪活动
  • 侵犯他人知识产权或隐私
  • 制作或传播色情、暴力或恐怖相关内容

3.2 滥用行为

  • 对平台进行恶意攻击（DDoS、暴力破解等）
  • 尝试绕过安全限制或速率限制
  • 利用漏洞获取未授权的访问权限
  • 批量注册账户或使用自动化工具套取免费额度
  • 共享、转售或二次分发 API 访问权限

3.3 有害内容

  • 生成虚假信息或深度伪造内容用于欺骗
  • 利用 AI 制作垃圾邮件或骚扰信息
  • 生成歧视性、仇恨性言论
  • 制作可能危害公共安全的内容

4. 资源使用限制

  • 遵守平台设定的速率限制和并发限制。
  • 不得发送过大或明显浪费资源的请求。
  • 不得使用本平台进行加密货币挖矿或类似的资源密集型活动。

5. API Key 安全

  • API Key 仅限注册本人使用，不得与他人共享。
  • 不得在客户端代码、公开仓库或前端应用中暴露 API Key。
  • 发现 API Key 泄露时应立即重新生成。

6. 内容审核

我们保留对 API 使用情况进行监控和审核的权利。发现违反本政策的行为时，我们可能会：

  • 发出警告通知
  • 临时限制或暂停服务
  • 永久终止账户
  • 在必要时向有关部门举报

7. 举报机制

如果您发现有用户违反本使用政策，请通过控制台内的联系方式向我们举报。我们将对举报内容进行调查并采取适当措施。

8. 政策更新

我们保留随时修改本使用政策的权利。修改后的政策将在本页面公布。继续使用本服务即表示您同意遵守更新后的政策。
