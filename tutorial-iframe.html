快速上手

几分钟内完成接入，开始使用 AnPin AI 的大模型 API 服务

1
注册账号并获取 API Key

点击右上角「控制台」按钮，使用邮箱注册 AnPin AI 账号。注册完成后登录控制台，在「API Keys」页面点击「创建 API Key」。

⚠️ API Key 仅在创建时显示一次，请立即复制保存到安全位置。如果遗失，需要重新创建。

你的 API Key 格式如下：

sk-anpin-xxxxxxxxxxxxxxxxxxxx
💡 注册后即赠送免费额度，可直接体验所有模型，无需充值即可开始。
2
安装 SDK

AnPin AI 同时兼容 Anthropic 和 OpenAI SDK，根据你的需求选择安装：

# Anthropic SDK（推荐）
pip install anthropic

# 或 OpenAI SDK
pip install openai
# Anthropic SDK
npm install @anthropic-ai/sdk

# 或 OpenAI SDK
npm install openai
go get github.com/anthropics/anthropic-sdk-go
3
发起第一次 API 调用

初始化客户端时只需替换 base_url 和 api_key，其他代码与官方 SDK 完全一致。

import anthropic

client = anthropic.Anthropic(
    api_key="sk-anpin-xxxx",
    base_url="https://api.anpin.ai"
)

message = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "Hello, introduce yourself in one sentence"}
    ]
)
print(message.content[0].text)
from openai import OpenAI

client = OpenAI(
    api_key="sk-anpin-xxxx",
    base_url="https://api.anpin.ai/openai/v1"
)

response = client.chat.completions.create(
    model="gpt-5.4",
    messages=[
        {"role": "system", "content": "You are a helpful assistant"},
        {"role": "user", "content": "Hello, introduce yourself in one sentence"}
    ]
)
print(response.choices[0].message.content)
curl https://api.anpin.ai/v1/messages \
  -H "Content-Type: application/json" \
  -H "x-api-key: sk-anpin-xxxx" \
  -H "anthropic-version: 2023-06-01" \
  -d '{
    "model": "claude-sonnet-4-6",
    "max_tokens": 1024,
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({
  apiKey: "sk-anpin-xxxx",
  baseURL: "https://api.anpin.ai",
});

const message = await client.messages.create({
  model: "claude-sonnet-4-6",
  max_tokens: 1024,
  messages: [
    { role: "user", content: "Hello, introduce yourself" },
  ],
});
console.log(message.content[0].text);
4
流式输出（Streaming）

对于需要实时展示回复的场景（如聊天界面），使用流式输出可以逐字显示模型回复，大幅提升用户体验。

import anthropic

client = anthropic.Anthropic(
    api_key="sk-anpin-xxxx",
    base_url="https://api.anpin.ai"
)

with client.messages.stream(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Write a quicksort in Python"}]
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

print()
import Anthropic from "@anthropic-ai/sdk";

const client = new Anthropic({
  apiKey: "sk-anpin-xxxx",
  baseURL: "https://api.anpin.ai",
});

const stream = client.messages.stream({
  model: "claude-sonnet-4-6",
  max_tokens: 1024,
  messages: [
    { role: "user", content: "Write a quicksort in JavaScript" },
  ],
});

for await (const event of stream) {
  if (event.type === "content_block_delta"
      && event.delta.type === "text_delta") {
    process.stdout.write(event.delta.text);
  }
}
5
在 Claude Code 中使用

Claude Code 是 Anthropic 官方的 AI 编程工具，支持 CLI、桌面端和 IDE 插件。配置 AnPin AI 只需两个环境变量：

# macOS / Linux
export ANTHROPIC_API_KEY="sk-anpin-xxxx"
export ANTHROPIC_BASE_URL="https://api.anpin.ai"

claude
# Windows PowerShell
$env:ANTHROPIC_API_KEY="sk-anpin-xxxx"
$env:ANTHROPIC_BASE_URL="https://api.anpin.ai"

claude
💡 也可以在 Claude Code 的设置面板中永久配置 API Base URL，无需每次手动设置环境变量。VS Code / JetBrains 插件用户可在插件设置中配置。
6
在第三方工具中使用

AnPin AI 兼容所有支持自定义 API 端点的第三方工具。常见集成方式：

✓ ChatBox / ChatGPT-Next-Web
✓ Cursor / Windsurf
✓ Open WebUI / LobeChat
✓ Cline / Aider / Continue
✓ LangChain / LlamaIndex
✓ Dify / FastGPT / Coze

通用配置方式：在工具的 API 设置中，将 API Base URL 设为 https://api.anpin.ai（Anthropic 格式）或 https://api.anpin.ai/openai/v1（OpenAI 格式），填入你的 API Key 即可。

7
充值与查看用量

在控制台「充值」页面选择套餐进行充值。在「个人中心」可查看实时余额、详细用量统计和调用日志。

所有模型按实际 Token 用量计费，汇率 ¥1 = $1。支持 Claude 全系列（Haiku / Sonnet / Opus）及 GPT 全系列模型。
